# HTML5 音乐播放可视化

教程来自慕课网。🎸

教程老师给的演示🔗  [passionate music](http://passionate.herokuapp.com/)

前期准备工作：

```
sudo npm install -g express-generator
sudo npm install -g supervisor
```

```
express -e h5-music
npm install
supervisor bin/www 
```

休息一下♨️
